# Trend Spy Bot

A real-time stock trend prediction application powered by machine learning.

## Features

- Real-time stock trend predictions
- Track up to 5 stocks simultaneously
- AI-powered forecasts for short-term price movements
- Modern, responsive UI built with React and TypeScript

## Technologies

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS
- Supabase

## Getting Started

### Prerequisites

- Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Step 1: Clone the repository
git clone https://github.com/doxncwfn/trend-spy-bot.git

# Step 2: Navigate to the project directory
cd trend-spy-bot

# Step 3: Install the necessary dependencies
npm install

# Step 4: Start the development server
npm run dev
```

## Development

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run lint` - Run ESLint
- `npm run preview` - Preview production build
